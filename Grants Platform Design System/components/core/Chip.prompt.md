Small toned pill for a label, status, verdict, or recommendation.

```jsx
<Chip label="Pursue" tone="good" />
<Chip label="Deadline: 2026-03-01" tone="neutral" />
```

Tones: `good` (pursue / approved / awarded), `warn` (maybe / revise / applied), `bad` (pass / error / dead), `info` (running / researching), `muted` (closed / inactive), `neutral` (bordered default), `brand` (brand-tinted).
